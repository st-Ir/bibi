About this Demonstration

This is a valid IDPF ePub3 which uses features than can only be seen
in AZARDI 17. It will load in all other Fixed Layout ePub3 readers but will not
display in the same presentation manner as when viewed on AZARDI.

This book has been FLO metadata configured for
landscape orientation and no page spread.
