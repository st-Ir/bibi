Famous Paintings

Using this book

  1. Scroll the Explore panel on the left to find a painting that interests you.
  2. Click on the picture icon to access the article
  3. Read about the painting and the artist
  4. Click on any painting in the information article to view the image full page.
  5. Click the Close X in the top left corner to return to the article. Or...
  6. Double-click the image to open it in a new zoom and pan window to explore the image in detail.

   

About the Content

The content for this book has been sourced from www.wikipedia.com. The text and image links are available on each page. If you click the links the source webpage will open in a new Remote Resources window.

This is an arbitrary collection of paintings to show how powerful and efficient navigation interactivity can be created in what is regarded as a linear reading format - ePub.

   

About this demonstration

This is a Fixed Layout ePub3 book demonstrating some of the extended features of the AZARDI ePub2/3 Reader.

AZARDI is designed to allow publishers to cost-effectively create ePub3 books for learning and education with extended and powerful interactive features that exceed the features of expensive Apps.

For more information see the Colophon and the APEX@IGP information page.
